# Приложение A. Ручной пример k-NN

Рассмотрим точку запроса Q=(4,2) и обучающие объекты в 2D. Расстояния до ближайших соседей: C (1.41, Blue), B (2.24, Red), A (3.16, Red). При k=3 мажоритарный класс — Red. Это иллюстрирует чувствительность метода к масштабу: без стандартизации признаки с большим разбросом доминируют в евклидовой метрике. В проекте все числовые признаки приведены к нулевому среднему и единичной дисперсии перед k-NN и SVM.

# Приложение B. Вывод градиента логистической регрессии

Для бинарного случая с сигмоидой σ(z)=1/(1+e^{-z}), z=w^T x, loss на одном объекте: L = -[y log σ(z) + (1-y)log(1-σ(z))]. Производная: dL/dz = σ(z)-y, откуда dL/dw = (σ(z)-y)x. Для softmax мультиклассовая форма аналогична с вероятностями p_k и one-hot меткой. L2-регуляризация добавляет член λw к градиенту. В sklearn оптимизация выполняется численно устойчивыми солверами lbfgs/saga.

# Приложение E. Глоссарий

**Accuracy** — доля верных ответов. **Precision** — точность положительного класса. **Recall** — полнота. **F1** — гармоническое средство precision и recall. **ROC-AUC** — площадь под ROC-кривой. **Cross-validation** — оценка обобщения на K фolds. **Transfer learning** — использование предобученных весов. **Fine-tuning** — дообучение всех или части слоёв на целевой задаче.
