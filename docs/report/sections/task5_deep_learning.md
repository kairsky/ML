# Задача 5. Свёрточные нейронные сети и трансферное обучение

## 5.1 CNN с нуля на CIFAR-10

**CIFAR-10** содержит 60 000 цветных изображений 32×32, 10 классов (транспорт, животные). Реализована сеть с **тремя свёрточными блоками**: в каждом два Conv2D 3×3, BatchNorm, ReLU, MaxPooling 2×2, Dropout 0.25. После блоков — GlobalAveragePooling, полносвязный слой 256, BatchNorm, Dropout 0.5, softmax. Число параметров ≈ **325 418** (см. `results/metrics/cnn_metrics.json`). **Аугментация данных:** горизонтальный flip, случайные сдвиги, zoom, jitter контраста — снижает переобучение и имитирует естественную вариативность сцены.

Обучение: **Adam**, lr=0.001, **30 эпох** (требование задания), early stopping, ReduceLROnPlateau, чекпоинты в `results/models/cnn_scratch/`. Итоговая модель — `results/models/cnn/cifar10_cnn_final.keras`. На тесте (быстрый прогон 3 эпохи ~62%; полное обучение 30 эпох ожидаемо **70–75%+** на CPU/GPU). Построены кривые обучения и confusion matrix с per-class метриками.

## 5.2 Архитектурная схема и параметры

Свёртки извлекают локальные паттерны (края, текстуры); глубина увеличивает рецептивное поле. MaxPooling обеспечивает инвариантность к малым сдвигам. BatchNorm ускоряет сходимость CNN так же, как в MLP. Схема архитектуры сохраняется в `results/plots/cnn/architecture.png` (или текстовый summary при отсутствии graphviz). Сравнение с MLP: CNN явно использует 2D-структуру, что критично для изображений.

## 5.3 Трансферное обучение ResNet18

Для эффективного использования предобученных представлений реализован **двухэтапный fine-tuning** на CIFAR-10 (изображения ресайзятся до 224×224, нормализация ImageNet):

1. **Freeze backbone** — заморожены все слои, кроме головы; обучается только линейный классификатор на 10 классов (~10 эпох).
2. **Unfreeze** — разморозка всей сети; дообучение с меньшим lr (~20 эпох).

Метрики (`results/metrics/transfer_learning_metrics.json`): test accuracy **~86%** (подвыборка train 10k для ускорения), **11.2M параметров**, время этапов head / finetune зафиксировано. ResNet18 с остаточными связями облегчает градиентный поток в глубоких сетях.

## 5.4 Интерпретируемость

Модуль `src/deep_learning/interpretability.py` поддерживает **Grad-CAM**: взвешенная сумма карт активаций последних свёрточных слоёв по градиентам целевого класса. Дополнительно возможны визуализации фильтров первого слоя и сетка «верно / неверно» с confidence (`plot_predictions_grid`). Это связывает практику с требованием объяснимости решений.

## 5.5 Сравнение CNN scratch vs transfer

| Критерий | CNN с нуля | ResNet18 transfer |
|----------|------------|-------------------|
| Точность (CIFAR-10) | Ниже при малых эпохах | Выше (~86% test) |
| Время обучения | Умеренное | Дольше (2 этапа) |
| Параметры | ~325K | ~11.2M |
| Данные | Нужны тысячи+ | Эффективен при ограниченных данных |
| Риск переобучения | Выше без аугментации | Ниже за счёт претрейна |
